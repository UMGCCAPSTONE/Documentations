# Gantt Chart Documentation #
This directory contains the project schedule and timeline used for the UMGC Capstone project. The schedule was created using GanttProject, a free and open‑source project management tool.
## Required Software ##
To view or edit the project schedule, you will need GanttProject.

**Why the .zip version is included**
- Portable and does not require installation
- Works on all major platforms
- Does not require administrator privileges
- Ensures reviewers use the same version used during development

This repository includes the recommended portable .zip distribution of GanttProject.

## How to Open the Gantt Chart ##
1. Download or extract the included ganttproject-3.3.3316.zip file.
2. Open the extracted folder.
3. Launch the GanttProject application:
   - Windows: ganttproject.exe 
   - macOS/Linux: run the appropriate startup script

In GanttProject, open the file: CMSC_Project_Schedule.gan  
The full project schedule, tasks, dependencies, and milestones will load automatically.

## Notes ##
- The .gan file is the authoritative version of the project schedule.
- Any updates to the timeline should be saved back into this directory.
- If GanttProject releases new versions, do not replace the included version unless the team agrees to update the schedule format.

# T-11: Database Schema

## Title
T-11: Database Schema

## Description
Design and implement the database schema for all MVP entities: users, inventory items, suppliers, menus, menu recommendations, and the recommendations log. Create and version control all migrations, ensuring they are idempotent and repeatable. Verify that the schema supports all required API operations and document the design.

## Acceptance Criteria
- [ ] Database tables created for users, inventory, suppliers, menus, menu_recommendations, and recommendation_history with appropriate data types
- [ ] All foreign key relationships defined, enforced, and documented
- [ ] Migrations are scripted, versioned, and repeatable (rollback-capable)
- [ ] Schema is documented with ERD diagram and/or table definitions accessible in the repository (README or wiki)
- [ ] Schema supports all MVP API operations (queries, inserts, updates identified in API spec)
- [ ] Migration scripts execute successfully in clean database environment
